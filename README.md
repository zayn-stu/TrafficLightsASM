## How to Run the Project

1. Install **DEEDS (Digital Electronics Education and Design Suite)** from:  
   https://www.digitalelectronicsdeeds.com

2. Open DEEDS and launch the **Micro Computer Emulator (d-McE)**.

3. Open the provided DEEDS project file (`.dmc`, `.dmce`, or `.mce`).

4. Open the Source Code Editor to view the DMC8 assembly program.

5. Assemble the program using:  
   **Project â†’ Assemble** or **Ctrl + F9**.

6. Run or Animate the simulation.

The system output can be observed on the LEDs.
